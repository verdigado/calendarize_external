===========
For Editors
===========

You can create records to schedule the import of your ICS urls or files.

* create an empty new folder in the page tree
* create a new "External calendar" record
* Fill in: title, url, optional note. Select how often the import should be scheduled

..  tip::

    Ask your admin, if no events are imported or errors occur.

The admin has to create scheduler entries for each scheduler interval.
